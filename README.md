# GrapeHub Performance Dashboard

GrapeHub is a comprehensive performance dashboard designed for sales and operational teams. It provides a suite of calculators and metrics tracking tools to optimize business processes.

## Features

- **Comercial Grape:** Track sales funnel metrics and performance.
- **Projects & Partners:** Manage projects and partner relationships.
- **Closer Calculator:** Analyze and project sales closing performance.
- **Gestor Calculator:** Manage traffic and operational metrics.
- **Admin Panel:** Comprehensive user and permission management.
- **Theme Support:** Fully integrated Light and Dark modes.

## Tech Stack

- **Frontend:** React, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend:** Node.js, Express (Full-stack setup).
- **Database:** Firebase Firestore, Neon DB (Postgres).
- **Authentication:** Firebase Authentication (Google Login).

## Getting Started

### Prerequisites

- Node.js (v18+)
- Firebase Project
- Neon DB Account

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/grapehub.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file based on `.env.example`.
4. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment

This application is designed to be deployed on Cloud Run or similar containerized environments.
